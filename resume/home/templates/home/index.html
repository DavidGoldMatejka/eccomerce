Task Completion Queue

Coded in Python, the Task Completion Queue is a way to publish tasks onto a queue. Published tasks are stored in a database and can be edited and deleted. Tasks are ordered by priority first, then by oldest to newest. Tasks can be removed from the queue by people who arn't the author so that it can be used by large teams. Tasks can be updated by anyone so that notes, change in priority or additional problems can be added on. Click on the title of this post to view the project.
